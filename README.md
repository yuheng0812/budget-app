# 記帳程式 Budget App

一個用 Python 寫的命令列記帳程式，可以新增支出、查看紀錄、計算總花費，並且關掉程式後資料不會消失。

## 功能

- 新增一筆支出（名稱、金額、類別、日期）
- 查看所有支出紀錄
- 顯示總花費
- 自動存檔，下次開啟資料還在

## 使用技術

- Python 3
- 檔案讀寫（CSV）
- 資料結構（List、Dictionary）

## 如何執行

1. 確認電腦有安裝 Python 3
2. 下載 `budget_app.py`
3. 在終端機執行：

python budget_app.py

## 操作方式

執行後會出現選單：
1. 新增支出
2. 查看紀錄
3. 顯示總花費
4. 離開
